Dashboard

Dashboard เป็นการที่เราทำให้ผู้ใช้รถสามารถ Approve หรือ Reject อุบัติเหตุได้และสามารถโดยการที่ Dashboard ทำขึ้นมาสามารถดูSOH(state of health) ของแบตเตอรี่นอกจากนี้ยังสามารถคุยกับโรงพยาบาลได้โดยการติดต่อผ่าน Dashboard หน้ารถโดยที่เราไม่จำเป็นต้องหยิบโทรศัพท์ขึ้นมาติดต่อ

Application Oceanpal Mobil(e)

Application Oceanpal Mobil(e) เป็น Application ที่จะบอกเราว่ามีอุบัติเหตุเกิดทีจุดไหนในแผนที่และยังจะมีหน้าที่เราจะสามารถในการกด Approve หรือ Reject ได้ ในกรณีที่คนขับรถมีคนนั่งมาด้วยและไม่สะดวกในการใช้งานตัว Dashboard รถยนต์ หรือ รถยนต์ ev นอกจากนี้ยังสามารถเช็คอุบัติเหตุที่เกิดล่าสุดได้อีกด้วย
